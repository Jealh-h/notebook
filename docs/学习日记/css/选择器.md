# css选择器

## 优先级

标签选择器---------------0001
类选择器|伪类:hover-------0010
ID选择器-----------------0100
行内样式-----------------1000  

- 标签选择器

```css
h1 {}
```

- 通配选择器

```css
* {}
```

- 类选择器

```css
.box-class {}
```

- ID选择器

```css
#id {}
```

- 标签属性选择器

```css
/* 选择href是'http://localhost:3000'的dom */
a[href='http://localhost:3000']{}
```

- 伪类选择器

```css

p:first-child {}
:nth-child(n) /*匹配第n个孩子*/
:nth-of-type(n) /*匹配第n个且类型是:前面的*/
```

- 伪元素选择器

```css
p::first-line{}
p::after{}
```

- 后代选择器

```css
p a {} /*选择p的后代且是a标签类型*/
```

- 子元素选择器

```css
article > p{} /*选择article的第一代孩子，且是p*/
```

- 相邻兄弟选择器

```css
h1 + p{} /*与h1相邻的p*/
```

- 通用兄弟选择器

```css
h1~p {} /*选择与h1同级的p*/
```
