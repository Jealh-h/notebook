# 创建

  1. 字面量创建
const reg = /\d/;
  1. 构造函数创建

    接受两个参数，第一个为匹配的模式，第二位标志位。

标志位有如下枚举：

  1. g 全局匹配：找到所有的匹配，而不是在第一个匹配停止
  2. i 忽略大小写
  3. m 多行匹配，\r、\n
  4. s 允许 . 匹配新的行
  5. u 将模式看作是 unicode 编码
  6. y 粘性匹配，每次匹配从上一次匹配的 index 开始匹配。
const ref = new RegExp(/\d/);
